# odin-recipes
This project is part of [The Odin Project] curriculum.  
It is a basic recipe website built to practice foundational **HTML skills** and **Git/GitHub workflow**.

## Features
- A main index page with links to individual recipe pages.
- Each recipe page includes:
  - Title and image of the dish
  - Description
  - Ingredients list
  - Step-by-step instructions
  - Navigation link back to the homepage

## Skills Demonstrated
- Structuring HTML with proper boilerplate
- Using headings, lists, links, and images
- Organizing files into directories (`recipes/` and `images/`)
- Working with relative paths for navigation and images
- Version control with Git (init, add, commit, push)
- Publishing a static site using GitHub Pages
